## Notes
Tailwind Config - extend fontFamily:
fontFamily: {
  display: ["var(--font-display)"],
  sans: ["var(--font-sans)"],
}

The application uses a playful, mobile-first design system similar to Duolingo.
We need 'Fredoka' for display fonts and 'Quicksand' for the body font to achieve a rounded, bouncy look.
